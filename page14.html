Application of the Parāvartya in Chemistry: Concentration Dilution Calculation

Problem: A chemist has 1697 mL of a concentrated acid solution and wants to divide it into smaller containers, each holding 13 mL. How many containers can be filled, and how much acid will remain? Solve this using the division method from the sutra.

Solution:

Given Values:

Total volume: 1697 mL

Volume per container: 13 mL

Apply the Sutra to Divide 1697÷13:

Step-by-Step Division Using the Sutra:

1. Divisor: 13 → Write as 10 + 3.

Reverse the sign of the deviation (3). Transformed divisor = (10 - 3). So, deviation = –3.

2. Dividend: 1697.

3. Perform division step-by-step:

Bring down the first digit (1). This is the first quotient digit.

Multiply by (−3): 1 × (-3) = -3.

Add to the next dividend digit (6): 6 + (-3) = 3. Next quotient digit = 3.

Multiply by (−3): 3 × (-3) = -9.

Add to the next digit (9): 9 + (-9) = 0. Next quotient digit = 0.

Multiply by (−3): 0 × (-3) = 0.

Add to the last digit (7): 7 + 0 = 7. Remainder = 7.

4. Result Quotient Q = 130, Remainder R = 7.

Interpretation: The chemist can fill 103 full containers, and 7 mL of acid will remain unused.

3. In Quadratic Equations: If the sum of numerators and the sum of denominators on both sides are equal, then 0 is one of the solutions. And, if the difference between numerators and denominators is equal on both sides, the solution is found by setting this difference to 0, which is another solution of the equation.

ax + b
cx + d
=
px + q
rx + s

Where, (p - r) = (a - c) and (b - d) = (s - q).

Example: 1

2x + 5
3x + 2
=
5x + 2
4x + 5

Steps:

Add numerators: (2x + 5) + (5x + 2) = 7x + 7

Add denominators: (3x + 2) + (4x + 5) = 7x + 7

Removing the constant term, the samuccaya x + 1 = 0

It is a sum of the numerator or denominator, (-1) is one root of the given equation.

Also, difference between numerator and denominator (N1 – D1) = (N2 – D2) = 0

Difference between N1 & D1: (2x + 5) - (3x + 2) = -x + 3 = -(x - 3)

Difference between N2 & D2: (5x + 2) - (4x + 5) = x - 3

Here, Samuccaya is –(x - 3), therefore 3 is another root of the equation.

4. In Special Cubic Equations: When the sum of terms in a cube equation (on either side) matches, the Samuccaya is the common term. The solution is found by setting this term to zero.

Example: (x - a)3 + (x - b)3 = 2(x - c)3 implies the Samuccaya is x - c and the solution is x = c.

Solve: (x - 1)3 + (x - 5)3 = 2(x - 4)3

1. Expand and simplify:

(x - 1)3+(x - 5)3 = 2(x - 4)3

x - 4 (neglecting the constant term) is common across the terms.

2. The Samuccaya is x - 4 = 0.

3. Solution: x = 4 is the root of the given equation.

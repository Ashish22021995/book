Example 2. Factorizing a Higher-Degree Polynomial

Given f(x) = x 3 -6x 2 + 11x - 6 = 0

1. Find the derivative:

f '(x) = 3x 2 -12x + 11

2. Use f(x) and f '(x) To identify roots. For example: Substituting x = 1 in f(x), we find f(1) = 0, so x = 1 is a root.

3. Perform synthetic division:

f(x) = (x-1)(x 2 - 5x + 6)

4. Factorize the quadratic:

x 2 -5x + 6 = (x - 2)(x - 3)

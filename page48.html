Example: Determining Successive Ionization Energies

The ionization energies (IE) of an element are related to its atomic structure. For a hypothetical element, the ionization energies (IE 1, IE 2, IE 3, IE 4) form an arithmetic progression (AP). Given the equations:

1
IE 1 ×, IE 2
+
1
IE 1 ×, IE 3
=
1
IE 1 ×, IE 4
+
1
IE 2 ×, IE 3

Find IE 1 , if IE 2 = IE 1 + 20KJ/mol, IE 3 = IE 1 + 40KJ/mol, IE 4 = IE 1 + 60KJ/mol.

Solution:

Step 1: Identify A, B, C, D

From the equation:

1
IE 1 ×, IE 2
+
1
IE 1 ×, IE 3
=
1
IE 1 ×, IE 4
+
1
IE 2 ×, IE 3

We identify:

A = IE 1, B = IE 2, C = IE 3, D = IE 4

Substitute:

B = A + 20,
C = A + 40,
D = A + 60

Step 2: Verify AP

B − A = 20,
C − B = 20,
D − C = 20.

Sum = 9 + 2(1) = 9 + 2 = 11

Step 3: Check divisibility:

11 ÷ 4 = not divisible.

Example 3: Multiplication by 12

Multiply 824 by 12.

Step 1: Write the number with a zero before and after:

824 → 08240824

Step 2: Apply the sutra step-by-step:

First step: Last digit 0 (ultimate) and second-last digit 4 (penultimate):

0 + 2(4) = 0 + 8 = 8

Write 8, no carryover.

Second step: 4 (ultimate) and 2 (penultimate):

4 + 2(2) = 4 + 4 = 8

Write 8, no carryover.
